## Node.js的前世今生

Node本身就是一個JavaScript執行環境，是對Google V8引擎的完美**封裝**。非瀏覽器環境執行的V8優化引擎。

Node.js框架基於事件驅動、非阻塞I/O模型，因而得以輕量和高效能。非常適合在分散式裝置終端上執行資料密集型的即時應用。

Node.js實際上已成為伺服器端的JavaScript指令碼語言開發平台。
